## Nox威力加强版

Virtuareal link所属虚拟艺人[诺莺Nox](https://space.bilibili.com/529249/channel/seriesdetail?sid=61060)，歌势。从Nox的社交媒体偷来了图换了皮肤。

本威力加强版在原版基础上功能更强，与手机端互通。仍然保留插件形式，无缝对接b站cookie。

### 新功能

- 播放b站，油管链接
- 歌单订阅更新
- 歌名提取与歌词搜索
- 换皮肤
- 歌单云备份、与手机版电梓播放器伪互通
- b站登录互动，点赞，三连，增加b站视频播放量
- ffmpeg处理音量均衡
